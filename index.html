Project 1: Information Extraction Data

Data Sources

This project extracts soccer player data from two popular sports statistics websites:

  • Transfermarkt - Market values and player statistics
  • FBref - Comprehensive player statistics from top 5 European leagues

HTML Data

Raw HTML data collected from each website:

Transfermarkt HTML Data Directory FBref HTML Data Directory

Extracted Tables

Processed data in CSV format:

Table A - Transfermarkt Data (CSV) Table B - FBref Data (CSV)

Data Schema

Both tables share the following schema:

  • ID - Unique identifier for each player
  • name - Player's full name
  • position - Playing position
  • age - Player's age
  • nationality - Player's nationality
  • current_club - Current team
  • market_value/matches_played - Value or games played depending on source
